# konstruktor
Sandbox for machine control simulation

# install

In root CMakeLists.txt setup correct path to vcpkg

set(CMAKE_TOOLCHAIN_FILE "C:/vcpkg/scripts/buildsystems/vcpkg.cmake" CACHE STRING "")

Download the following libraries to ThirdParty

- doxygen-1.17.0 (https://www.doxygen.nl/)
- imgui (https://github.com/ocornut/imgui/branches)
- tinygltf (https://github.com/syoyo/tinygltf)
